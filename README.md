# listen-now-desktop

listen-now 的桌面版

采用vue全家桶+electron
## 项目结构

- src 
    - main
        - mainUtil 存放主进程的操作
    - renderer
        - asserts 静态资源文件
        - components 通用组件
            - common
        - pages 各个页面以及页面单独组件
        - renderUtil 渲染进程所用到的操作
        - router 路由表
        - store
